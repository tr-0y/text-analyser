# text-analyser
A Python text analysis tool that processes input text to generate detailed linguistic statistics. Includes word and character counts, lexical diversity, hapax legomena count, word-length distribution, even most common bigrams and phrases.

# Features
Total word and character counts  
Unique word count  
Lexical diversity
Hapax legomena count
Average sentence and word length
Longest words in the text  
Word-length distribution  
Most frequent words  
Most common phrases
